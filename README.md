# NeuralNetworks_for_ADCS
Arduino Neural Network for Attitude Determination and Control Systems

Content:
- Folder "Libraries": the libraries used in this project, I2Cdevlib by jrowberg
- Folder "Sensor Readings + Output (No NN)": scripts for reading MPU6050 sensor data and processing it for obtaining the angle and acceleration
- Folder "PID examples": example scripts where a PID was used
- Folder "Reference": example script of a NN that works similarly to what we want to achieve
- File "NN.ino": example script of a NN 
- File "P2.ino": This project's script
- File "README.md"
